# Changing the Colors of the Interface

In bigger environments with test instances, it's helpful to change the colors of the Syncer GUI to tell the environments apart.

That's possible in the [local_config.py](lcl_config.md). You can use the variable `STYLE_NAV_BACKGROUND_COLOR` to set an HTML color for the background, `STYLE_NAV_LINK_COLOR` to change the color of the navigation links and finally `HEADER_HINT` to add a free text to the header. That could be the environment name.
